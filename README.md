# knulli-bezels

A collection of bezels for Knulli.

I have not created any of the bezels in this collection. This is merely a compilation of bezels from different creators that I combined into a set for personal use.

It consists of bezels from the following sources:

* [Batocera Bezels](https://github.com/batocera-linux/batocera-bezel)
* ED Bezels (source unknown)
* [Handheld bezel collection by nvitaterna](https://github.com/nvitaterna/batocera_4_3_handheld_bezels), compiled from bezels by drkhrse and antiKk
