# mugwomp93 bezels

Bezel sets by [mugwomp93](https://github.com/mugwomp93) to be used with [KNULLI](https://knulli.org).

The mugwomp93 bezel sets contain bezels for multiple purposes, such as

* bezels which are optimized for integer scaling (no additional settings needed, by simply using the bezel, **integer scaling is done implicitly!**)
* bezels which have grids for efficient LCD/CRT effects
* bezels which have both, implicit integer scaling **and** grids for LCD/CRT effects
* bezels which have **neither** integer scaling **nor** grids

The mugwomp93 sets are still a work in progress - not every system has a bezel of every type for every possible device screen, yet. To get an overview of the mugwomp93 bezel collection, please check the table below.
